Deploy Your Site in 20 Minutes

A beginner-friendly guide to hosting with GitHub Pages.

1 Create a GitHub Account

Go to GitHub.com and sign up for a free account. This will be the home for your website's code.

2 Create a New Repository

  1. Click New on the Repositories sidebar. 00:33
  2. Name your repository (e.g., my-website). 00:39
  3. Ensure visibility is set to Public. 01:04
  4. Check the box to Add a README file. 01:09
  5. Click Create repository.

3 Upload Your Content

  1. Inside your repo, click Add file > Upload files. 01:36
  2. Drag and drop your index.html file. 01:40
  3. Scroll down and click Commit changes. 01:46

4 Activate GitHub Pages

  1. Navigate to Settings (top tab) > Pages (left menu). 02:00
  2. Under "Branch", select the main branch and click Save. 02:03

5 Success! View Your Site

GitHub will now generate your live link. It usually looks like this:

https://[your-username].github.io/[repo-name]/

Note: It may take up to 2 minutes for the site to appear the first time. 02:22

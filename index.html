Step 1: Create a GitHub Account

Step 2: Create a New Repository

  1. Go to Repositories and click New. [00:00:33]
  2. Name your repository (e.g., my-website). [00:00:39]
  3. Set visibility to Public. [00:01:04]
  4. Add a README file. [00:01:09]
  5. Click Create repository.

Step 3: Upload Your Files

  1. Click Add file > Upload files. [00:01:36]
  2. Upload your index.html file. [00:01:40]
  3. Click Commit changes. [00:01:46]

Step 4: Configure GitHub Pages

  1. Go to Settings > Pages. [00:02:00]
  2. Select the main branch and click Save. [00:02:03]

Step 5: View Your Site!
